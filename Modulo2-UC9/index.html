Atributos


class - aplicada para formatar varios elementos


id - aplicada para formatar um elemento


style - permite incluir estilos de CSS


lang - define o idioma principal


title - Define o titulo do elemento


alt - define um texto alternativo, muito utilizado para descrição de imagens


hidden - oculta o elemento


align - define o padrão de alinhamento como : right(direita),center(centralizado), left(esquerda) e justify(justificado)


width - determina a largura do elemento

height - define uma altura para o elemento
